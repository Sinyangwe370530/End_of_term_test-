# Student Profile Program

## Project Description
This is a simple C program that displays a student's profile, including their name, age, and GPA. The program prints the information in a neat, formatted style and ends with a clear footer message.

**Features:**
- Displays the student's name, age, and GPA
- Uses formatted output for better readability
- Shows a program-end message 
- Demonstrates basic C concepts like variables, data types, and `printf`
